This tutorial contains 3 Part: Gene, SNP, Protein, Video, Download.

Gene

In Gene part, the available input types are Gene symbol, ENSG (Ensembl gene ID) and gene coordinate (chromosome and position). By providing the gene symbol or ENSG ID,




By clicking these links, the website will be redirected to other external databases



two tables includes symbol, ENSG ID, gene coordinate, gene full name, gene description, corresponding Isoform ID and PDB ID would be shown. An additional GWAS catalog annotation on this gene would also be shown on the bottom of the page.





A genome browser would also be shown together with the introns and exons corresponding to that gene, as well as the recombination rate and GWAS catalog for relevant SNPs. The position range of the genome browser would be +/- 25000 bp of the start/end site of that gene and therefore any closed gene on that chromosome could also be visualized.




Another isoform browser is also available, which shows different isoforms (ENST) of that gene.



By clicking the isoform on the isoform table, the structure of that isoform will be shown solely.




At the bottom of the page, there is a SIFT and Polyphen Prediction table




Additionally, a table contains the information of SNPs on this gene would be shown. The table initially provides the SNPs only in the Exomes. You can change is by clicking the Genomes option




By clicking the SNP in the SNP table, it will pop out a new webpage. The information and the annotation of that SNP will be shown on the new page. The position of that SNP will be shown on the two genome browsers as a red vertical line. The remains of the figures are the same with the genome browser above. For the following available steps for SNP, please refer to SNP part.








If user provides the range of the position and select the chromosome,




a genome browser include all the genes within the range, together with GWAS catalog SNP information and recombination rate will be shown.




SNP

In this part, the available input types are dbSNP ID (rsID) or base position (chr:position). By providing the SNP ID or its position,




users are able to view its information (SNP table), annotation (Variant Effect Predictor(SIFT and Polyphen), Population frequency table) as well as the corresponding gene and isoform information.






Similarly, two genome browsers would also be shown with the corresponding SNP representing as a red line.




Additionally, one novel feature of this application is that the SNP could be visualized in the 3D protein structure. By clicking on the link of the PDBID, the corresponding 3D protein structure could be visualized with highlighting the selected SNP coding amino acid. For the operations in viewing 3D protein structure, please refer to Protein part.

Protein

Users can input an PDB ID from the main page.



Then a new page for the specific PDB entry will appear. In the new page, there will be a link to see the 3D structure of the PDB entry. In addition, there will be a SNP table at the bottom page which includes all the SNP position available for the PDB.







Users can enter SNP ID or exact risidue number and chain number for a specific SNP residue on the protein, and a new link to the PDB entry which highlights the SNP position on the 3D structure will appear on the page. In addition, users can also click on a SNP ID in the SNP table, and easily highlight it in the 3D structure.







Finally, users can choose the range (from 3 to 9 angstrom) for nearby SNPs via the advanced settings, then a new table of residues within a specific angstrom of the selected SNP residue will be shown.


Its last column will indicate if the residue is a SNP or not. Users can view the position of the nearby residue(in blue color) by clicking the button.
